Download Page.

This page contains desktop and commandline versions of the tools in BioImage Suite web. BioImage Suite Web also works directly in your browser and can be accessed from this webpage.

Current Version

Version 1.0, Dec 12, 2018

  1. Desktop Applications
    • Microsoft Windows installer (~ 140 MB)
    • MacOS zipped .app file (~ 140 MB)
    • Linux (Ubuntu) zip file (~ 140 MB)

    The desktop applications are based on (and include) Electron.

  2. Command Line Tools
    • All-platform command line tools package. (~ 10 MB)
    • All-platform command line tools package with test data. (~ 50 MB)

    Please note that these distribution should work on both Windows, Mac OS and Ubuntu. In all three cases you will need to install Node.js v8.x or v10.x LTS.

Developement Version 1.1.0a, Jan 2019

  • Command Line Tools

    • Please note that these distribution should work on both Windows, Mac OS and Ubuntu. In all three cases you will need to install Node.js v10.x LTS.

    To install, use npm. To install this globally (recommanded), open a console and then type: 

                sudo npm install -g biswebnode

    (On Ms-Wndows omit the sudo part).


    To run any command line tool then simply type:

     biswebnode modulename

    or

     biswebnode

    to get a list of all the modules.

    To uninstalll type sudo npm remove -g biswebnode.

    You may obtain the source code for this application from our main github repository and from the plugin repository.

    If you are looking for the desktop C++/TCL of legacy version of Yale BioImage Suite it may be downloaded from this webpage.
